# Dream Job

This project is a training web application built in Java using Spring (Spring MVC and Spring Boot).

The main goal of the project is to understand how web application architecture works.

## Description

The application simulates a job platform where users can interact with vacancies and resumes.

There are two types of users:
- Candidates — publish resumes
- Recruiters — create job vacancies

## Features

- Candidates can:
    - create and publish resumes
    - respond to job vacancies

- Recruiters can:
    - create and manage vacancies
    - invite candidates to open positions

## Purpose

This project helps to learn:
- basic web application architecture
- working with data models
- interaction between users
- application structure at a high level

## Technologies

- Java
- Spring MVC
- Spring Boot
